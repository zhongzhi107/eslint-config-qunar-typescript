# eslint-config-qunar-typescript

## 安装
```
npm i eslint-config-qunar-typescript
```

## 使用

在 eslint 配置文件中继承 `eslint-config-qunar-typescript` ：

```js
// .eslintrc.js
module.exports = {
  extends: ['eslint-config-qunar-typescript']
};
```

```json
// .eslintrc
{
  "extends": ["eslint-config-qunar-typescript"]
}
```
